# evolution
## Launch
### How to
0. Install python 3.12
1. Install libraries
   `pip install -r requirements.txt`
2. Launch main.py
   `python3.12 main.py`
### Flags
- `--mut-alg <0-2>`: specify mutation algorithm. 2 is default
- `--load`: takes no arguments. If the flag is given, autosave will be loaded automatically at the start.
